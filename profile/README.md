## 📋 Overview

<a href="#clickCharts">ClickCharts</a> is a flowchart and diagram creation application for Mac — providing a drag-and-drop shape library, multiple diagram types, and export options for creating process documentation, organizational charts, UML diagrams, and any visual representation of information that structured diagram tools produce.

The <a href="#clickCharts">drag-and-drop diagram canvas</a> builds diagrams by selecting shapes from the library, placing them on the canvas, and connecting them with lines and arrows — the visual construction approach that makes diagramming accessible without specialized training. Shapes resize, reposition, label, and connect through direct manipulation, and the canvas scales to accommodate diagrams from small process flows to large complex organizational charts. The <a href="#clickCharts">shape library breadth</a> covers the standard shape vocabulary for different diagram types — basic flowchart shapes (process, decision, terminator, data), UML shapes for class and sequence diagrams, network topology shapes, organizational chart boxes, entity relationship diagram notation, and general-purpose shapes for custom diagram needs.

The <a href="#clickCharts">multiple diagram type support</a> handles the variety of diagram formats that different communication needs require — process flowcharts for documenting procedures, organizational charts for displaying reporting structures, UML class diagrams for software architecture, swimlane diagrams for cross-functional process documentation, data flow diagrams for information system analysis, mind maps for brainstorming and concept organization, and network diagrams for IT infrastructure documentation. The <a href="#clickCharts">connection line tools</a> draw lines and arrows between shapes with configurable styles — straight lines, elbow connectors, curved connections, and arrowhead options that communicate the direction and relationship type between connected elements.

<a href="#clickCharts">Text and label formatting</a> configures font, size, color, and alignment for shape labels and standalone text elements — professional-looking diagrams that communicate clearly without requiring design expertise. <a href="#clickCharts">Color and style customization</a> applies fill colors, border styles, line weights, and opacity to shapes and connections — visual differentiation between diagram elements that aids comprehension.

<a href="#clickCharts">Export to multiple formats</a> produces PDF for print and sharing, PNG and JPG for embedding in documents and presentations, and other formats for different distribution contexts. <a href="#clickCharts">Template starting points</a> provide pre-built diagram structures for common use cases — opening a flowchart template, org chart template, or UML template with example structure in place as a starting point for customization.
